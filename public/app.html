        Fundamentos
        
    Lógica Matemática Aplicada ao Sistema
    
        
    Como o LogicPC Expert representa conhecimento técnico como fórmulas lógicas.
    

        
    
          
    
            
    ① Proposições
    
            
    Cada pergunta é uma proposição lógica que pode ser verdadeira (V) ou falsa (F).
    
            
    p = "o computador liga"
q = "a fonte faz barulho"
r = "existe tela azul"
    
          
    

          
    
            
    ② Operadores
    
            
    O sistema utiliza AND (∧) e OR (∨) para combinar proposições.
    
            
    ¬p ∧ ¬q   → não liga E sem barulho
p ∧ ¬imagem → liga E sem imagem
aquece ∧ desliga → AND lógico
    
          
    

          
    
            
    ③ Regras IF-THEN
    
            
    O conhecimento é representado como regras de produção.
    
            
    IF ¬liga ∧ ¬barulhoFonte
THEN problema = Fonte

IF aquece ∧ desligaSozinho
THEN problema = Superaquecimento
    
          
    

          
    
            
    ④ Inferência
    
            
    O motor avalia cada regra e deduz a hipótese mais provável.
    
            
    Fatos: { ¬liga, ¬barulho }
Regra: ¬liga ∧ ¬barulho → Fonte
∴ Conclusão = "Fonte de alimentação"
